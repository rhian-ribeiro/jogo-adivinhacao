# 🎲 Jogo da Adivinhação

Projeto desenvolvido para praticar JavaScript.

## Funcionalidades

- Geração de número aleatório
- Histórico de palpites
- Validação de entrada
- Dicas de número alto ou baixo
- Contador de tentativas
- Botão para reiniciar o jogo

## Tecnologias

- HTML
- CSS
- JavaScript
